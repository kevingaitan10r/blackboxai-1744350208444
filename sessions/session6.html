Sesión 6: Evaluación de Usabilidad

Métodos de Evaluación de Usabilidad

1. Tests de Usabilidad

Observación directa de usuarios interactuando con la interfaz.

Componentes Clave

  • Participantes representativos
  • Tareas específicas
  • Observación sistemática
  • Recopilación de datos

Métricas

  • Tasa de éxito
  • Tiempo de completación
  • Tasa de error
  • Satisfacción subjetiva

2. Evaluación Heurística

Análisis experto basado en principios establecidos de usabilidad.

Principios de Nielsen

  • Visibilidad del estado del sistema
  • Coincidencia con el mundo real
  • Control y libertad del usuario
  • Consistencia y estándares

Proceso de Evaluación

  • Inspección sistemática
  • Identificación de problemas
  • Clasificación de severidad
  • Recomendaciones

3. Análisis de Datos

Recopilación y análisis de datos cuantitativos y cualitativos.

Datos Cuantitativos

  • Analytics
  • Mapas de calor
  • Tasas de conversión
  • Métricas de rendimiento

Datos Cualitativos

  • Feedback de usuarios
  • Entrevistas
  • Observaciones
  • Comentarios

Ejemplo Práctico: Evaluación de un E-commerce

Test de Usabilidad

Tarea: Completar una compra

Participantes: 10 usuarios

Resultados:

  • 70% completó la tarea sin ayuda
  • Tiempo promedio: 3.5 minutos
  • Principal problema: proceso de checkout

Evaluación Heurística

Problemas Identificados:

  • Navegación poco clara
  • Falta de feedback en formularios
  • Inconsistencia en botones

Análisis de Datos

Métricas Clave:

  • Tasa de abandono: 35%
  • Tiempo en página: 2.5 min
  • Tasa de conversión: 2.8%
