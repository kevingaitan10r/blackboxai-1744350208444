# UI/UX Design Learning App

## Project Overview
The UI/UX Design Learning App is a web-based application designed to help users learn the fundamental principles of User Interface (UI) and User Experience (UX) design. It features multiple sessions covering a variety of topics, ensuring that learners can gain a comprehensive understanding of the subject matter. The app employs an intuitive design framework with animated transitions to enhance user engagement.

## Installation
To run the UI/UX Design Learning App locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone <repository_url>
   ```

2. **Navigate to the project directory:**
   ```bash
   cd <project_directory>
   ```

3. **Open the `index.html` file in your web browser:**
   ```bash
   open index.html
   ```
   or simply double-click the `index.html` file.

## Usage
Once the app is open in your web browser, you can explore the various learning sessions available. Click on the "Comenzar" button under each session to start learning about specific topics. Progress is tracked, and you can see your completed sessions and scores in the application.

## Features
- **Multiple Learning Sessions:** The app includes 8 different sessions, each focusing on a key aspect of UI/UX design.
- **User Progress Tracking:** The application allows users to track their progress through sessions, showing which sessions they have completed and their respective scores.
- **Responsive Design:** The app is built using Tailwind CSS, ensuring a responsive and modern user interface suitable for various screen sizes.
- **Animated Interactions:** Smooth animations enhance the user experience when navigating and interacting with elements within the app.

## Dependencies
The UI/UX Design Learning App utilizes the following dependencies:

- [Tailwind CSS](https://tailwindcss.com/): A utility-first CSS framework for styling.
- [Font Awesome](https://fontawesome.com/): A library for scalable vector icons.

These dependencies are included via CDN links in the `index.html` file.

## Project Structure
The project consists of the following structure:
```
├── index.html           # Main HTML file for the application
